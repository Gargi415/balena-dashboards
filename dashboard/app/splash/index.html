To get started, set the following Device Enviroment Variables. From the device summary page, select Device Variables for environment variables. Click Add variable, select the appropriate service if necessary, add a name and and a value, and click Add.

Name Value
SLIDE_COUNT The number of slides
SLIDE_0 The full dashboard URL
TIME_0 The number of seconds to display
SLIDE_[...] Up to 10 slides/timeouts can be added
